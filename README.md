# 🌾 Farmer Crop Management System

## 📌 Description
This project is a Java-based application that helps farmers manage crop data and connect with companies for selling crops.

## 🛠 Technologies Used
- Java (Swing GUI)
- MySQL Database
- JDBC Connectivity

## ✨ Features
- Farmer crop management
- Company registration
- Customer order system
- Matching farmers with companies
- Report generation

## 🗄 Database Setup
1. Open MySQL
2. Import the file: database/database.sql
3. Run the queries

## ▶️ How to Run
1. Open project in Eclipse
2. Run LoginForm.java
3. Use system features
